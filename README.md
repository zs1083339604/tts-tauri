# Tauri + Vue 3

使用Tauri开发的微软配音工具

## 注意事项

- 第一次使用Tauri发布公开软件，据说Tauri的兼容性不好，有问题请提交。
- 只能保证在Windows系统上的正常使用，因为我没有其他电脑系统做测试(～￣▽￣)～
- 音频使用base64传输，文本过多时会有性能问题
